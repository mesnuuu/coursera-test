Hello Faris, thanks for the screeenshot. I forgot to mention that whenever git config pops out, all you have to do is do something like this:
1 - type and enter the following at the command prompt/terminal: git config --global user.email "your email of choice goes here in the form of myemail@somecompany.com"

2 - type and enter the following at the command prompt/terminal: git config --global user.name "Just type your name here"

3 - I suggest you to redo the step six from the main series of steps and onward from here again starting with git commit -m "relevant comments go here regarding project"

4 - If step 3 triggers a popup window for you to sign in to github.com, go ahead and do so. Just so you know, if that fails for whatever reason, git will ask you about github account information at the terminal/command prompt. Cheers.
